# Sistem Case-Based Reasoning (CBR) Putusan Pengadilan

Repositori ini berisi implementasi sistem *Case-Based Reasoning* (CBR) sederhana berbasis Python untuk menganalisis putusan pengadilan dari Mahkamah Agung Republik Indonesia.

## Struktur Repositori
- `data/` : Folder untuk menyimpan dataset putusan
  - `pdf/` : File PDF putusan asli
  - `raw/` : File teks hasil ekstraksi dari PDF
  - `processed/` : Data terstruktur hasil ekstraksi metadata dan fitur (.csv/.json)
  - `eval/` : Data pengujian dan hasil evaluasi metrik
  - `results/` : Hasil prediksi sistem
- `notebooks/` : Jupyter Notebooks untuk setiap tahapan (Tahap 1 s/d 5)
